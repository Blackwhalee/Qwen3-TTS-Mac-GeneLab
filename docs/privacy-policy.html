隐私政策

适用于 macOS 应用「黑鲸自定义 TTS」（Bundle ID：com.blackwhale.YujieTTS）。最后更新日期：2026 年 4 月 1 日。

1. 概要

本应用主要在您的 Mac 本地完成文本转语音（TTS）推理。我们不会将您的朗读文本或生成的语音用于广告画像，也不出售个人数据。

2. 本机处理与存储

  • 语音合成模型、Python 运行环境与项目代码在首次启动时下载至您用户目录下的应用数据文件夹（Apple 沙盒内），并在本地运行。
  • 生成历史、音频片段、语音相关配置等默认保存在本机应用支持目录内，不上传至我们运营的服务器（我们未为此 App 部署用于收集用户内容的中央后端）。
  • 内购相关的「首次免费是否已使用」「剩余次数」等状态使用系统提供的本地存储（UserDefaults）记录，用于兑现购买与免费策略。

3. 网络访问

  • 首次引导：可能从 GitHub Releases 等平台下载打包好的环境与源码归档（仅用于安装运行所需文件）。
  • 模型与依赖：可能通过 Hugging Face Hub 等渠道下载或更新机器学习模型与相关资源；该过程受第三方服务条款与其隐私政策约束。
  • App 内购买：由 Apple 的 App Store / StoreKit 处理；我们不直接接收您的银行卡信息。

4. 跟踪与第三方 SDK

本应用隐私清单中声明不启用跨轨跟踪；未集成用于广告追踪的第三方分析 SDK。若未来版本有变更，将更新本页面与 App 内说明。

5. 儿童

本应用面向普通用户，不刻意以未满 13 周岁儿童为主要受众。若您为监护人且对数据处理有疑问，请通过下方方式联系我们。

6. 变更

我们可能随版本更新修订本政策；重大变更时在合理范围内通过应用更新说明或本页顶部的「最后更新日期」提示。

7. 联系我们

有关本隐私政策或本应用数据实践的问题，请通过 App Store 产品页上公布的支持链接或开发者网站中提供的方式联系开发者。

开源与问题反馈：GitHub Issues（可选）。
