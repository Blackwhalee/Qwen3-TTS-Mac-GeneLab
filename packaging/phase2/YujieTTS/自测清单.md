# 黑鲸自定义克隆 TTS — 自测清单

自测时每项前打 `[x]`。发现问题记下**复现步骤 + 截图/日志**。

## 0. 环境与构建

- [ ] **仅 Apple Silicon**（工程为 `arm64`）  
- [ ] Xcode 打开 `YujieTTS.xcodeproj`，**Run** 为 **Debug**（日常自测）或 **Release**（贴近上架）  
- [ ] **上架前模拟：** 已执行 `prepare_bootstrap_resources.sh`，Resources/bootstrap 内有两枚 `.tar.gz`（`.gitignore` 不提交）

## 1. 首次启动 / 引导（冷启动）

**每次测前先清数据（任选其一）：**

- 沙盒：`~/Library/Containers/com.blackwhale.YujieTTS/Data/Library/Application Support/YujieTTS`  
- 或访达「前往文件夹」搜索 `YujieTTS` 删除同名数据目录  

清单：

- [ ] 启动后出现 **Bootstrap**，文案含「黑鲸自定义克隆TTS」  
- [ ] 有 **bootstrap 包**时：提示类似 **「应用内资源，不经外网下载」**，环境与源码解压完成，无明显长时间假死  
- [ ] 无 bootstrap、仅有 `~/Qwen3-TTS-Mac-GeneLab/dist/` 时：仍能从 dist 安装  
- [ ] 引擎就绪后进入 **主界面**，无卡在「检查环境」  

## 2. 模型下载（首次）

- [ ] 引导结束后进入 **模型下载**阶段（若本地无缓存）  
- [ ] 网络正常时 **snapshot_download** 能完成（体积大，耐心等待）  
- [ ] **弱网/失败**：是否出现可读错误；**重试**可用  
- [ ] （可选）第二次启动 **不再**重复下模型  

## 3. 核心合成

- [ ] 输入短句 → **生成** → 成功返回音频，可 **播放**  
- [ ] **历史记录**写入、重启 App 仍可见（抽样测一条）  
- [ ] **参数/声音面板**调整后再生成，行为符合预期（无明显崩溃）  

## 4. 设置与菜单

- [ ] **设置**（`Settings` 场景）可打开，选项保存正常  
- [ ] **内购相关**（若有）：首次免费 / 次数 / 恢复购买 / Paywall 展示与文案无明显错别字（沙盒账号按需在 StoreKit 配置）  

## 5. 体验与边界

- [ ] 窗口缩放、最小尺寸无布局崩坏  
- [ ] **长时间空闲**后再生成仍正常  
- [ ] **Console**（子系统 `com.blackwhale.YujieTTS`）无频繁异常刷屏  

## 6. 贴近上架（账号具备时）

- [ ] **Product → Archive** → **Validate App** 通过  
- [ ] `sign_python_env_for_appstore.sh` 按文档在需要时已执行  
- [ ] TestFlight 或分发包在**干净用户**下重复 **§1～§3**
